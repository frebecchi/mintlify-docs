## Account Management

Once accounts are added, you can manage them through the Accounts page.

## Viewing Accounts

### Account List

The accounts page shows all connected accounts:
- Phone number or username
- Session name
- Connection status
- Profile picture
- Created date
- Actions menu

### Account Status

**Connected:**
- Account is healthy
- Can send messages
- Green indicator

**Disconnected:**
- Account needs reconnection
- Cannot send messages
- Red indicator

**Checking:**
- Health check in progress
- Temporary status

## Account Actions

### Test Connection

Verify account is working:
1. Click account menu (three dots)
2. Select **"Test Connection"**
3. Health check runs
4. Status updates

<Info>
  Use test connection to verify account health before creating campaigns.
</Info>

### Reconnect Account

If account is disconnected:
1. Click **"Reconnect"** button
2. Follow authentication steps
3. Re-authenticate account
4. Account reconnects

### Delete Account

Remove account from Reach:
1. Click account menu
2. Select **"Delete Account"**
3. Confirm deletion
4. Account is removed

<Warning>
  Deleting an account removes it from all campaigns using it. Update campaigns before deleting.
</Warning>

## Account Health

### Automatic Checks

Reach automatically checks account health:
- Regular monitoring
- Status updates
- Health indicators
- Connection verification

### Manual Checks

Test account connection manually:
- Use "Test Connection" action
- Immediate status update
- Verify account is working
- Check before campaigns

## Multiple Accounts

### Managing Multiple Accounts

When you have multiple accounts:
- Distribute message load
- Use different accounts for different campaigns
- Backup accounts available
- Monitor all account health

### Account Selection

When creating campaigns:
- Select which account to use
- Each campaign uses one account
- Account must be connected
- Can change account per campaign

## Best Practices

1. **Monitor Health**: Regularly check account status
2. **Test Connections**: Verify accounts before campaigns
3. **Use Multiple**: Distribute load across accounts
4. **Keep Connected**: Reconnect disconnected accounts promptly
5. **Secure**: Keep account credentials secure

## Troubleshooting

### Account Disconnected

If account shows disconnected:
1. Check Telegram app status
2. Verify phone number
3. Try reconnecting
4. Check for service issues
5. Contact support if needed

### Health Check Fails

If health check fails:
1. Verify account is valid
2. Check Telegram status
3. Try reconnecting
4. Review error messages
5. Contact support if persistent
