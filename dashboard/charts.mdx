## Chart Overview

The Reach dashboard includes several interactive charts and visualizations to help you understand your outreach performance at a glance.

## Message Performance Chart

The Message Performance Chart displays message trends over time.

### What It Shows

- **Messages Sent**: Daily, weekly, or monthly message volume
- **Delivery Rate**: Percentage of messages successfully delivered
- **Response Rate**: Percentage of messages that received responses
- **Trend Lines**: Visual indicators of performance trends

### Reading the Chart

- **X-Axis**: Time period (days, weeks, or months)
- **Y-Axis**: Number of messages or percentage
- **Lines**: Different colored lines represent different metrics
- **Tooltips**: Hover over data points to see exact values

### Use Cases

- Track message volume trends
- Identify peak sending periods
- Monitor delivery and response rates
- Compare performance across time periods

## Lead Funnel Chart

The Lead Funnel Chart visualizes the conversion process from leads to deals.

### Funnel Stages

1. **Total Leads**: All leads in your system
2. **Leads in Campaigns**: Leads included in active campaigns
3. **Leads Responded**: Leads who responded to messages
4. **Leads Converted**: Leads converted to deals

<Note>
  Screenshot placeholder: Add a visual funnel diagram showing the conversion stages
</Note>

### Understanding the Funnel

- **Width**: Represents the number of leads at each stage
- **Narrowing**: Shows where leads drop off in the process
- **Conversion Rates**: Percentage of leads moving to the next stage

### Optimization Opportunities

Use the funnel to identify:
- Stages with high drop-off rates
- Conversion bottlenecks
- Areas for improvement
- Successful conversion paths

## Campaign Performance Chart

The Campaign Performance Chart compares performance across your campaigns.

### Metrics Displayed

- **Campaign Completion**: Percentage of contacts completed
- **Messages Sent**: Total messages per campaign
- **Response Rate**: Response rate by campaign
- **Deals Created**: Number of deals generated from each campaign

### Chart Types

The chart may display:
- **Bar Chart**: Comparing metrics across campaigns
- **Line Chart**: Showing trends over time
- **Pie Chart**: Showing distribution of performance

### Identifying Top Performers

Look for campaigns with:
- High completion rates
- Strong response rates

## Deal Distribution Chart

The Deal Distribution Chart shows how deals are distributed across pipeline stages.

### What It Shows

- **Deals by Stage**: Number of deals in each pipeline stage
- **Deal Values**: Total value of deals in each stage
- **Stage Progression**: Visual representation of deal flow

### Pipeline Visualization

- **Columns**: Each column represents a pipeline stage
- **Height**: Number of deals or total value
- **Colors**: Different colors for different stages
- **Labels**: Stage names and deal counts

### Insights

Use this chart to:
- Identify bottlenecks in your pipeline
- See where deals accumulate
- Track stage conversion rates
- Optimize pipeline stages

### Source Categories

- **Campaign Sources**: Leads from specific campaigns
- **Manual Import**: Leads imported via CSV
- **API**: Leads added via API integration
- **Other**: Other sources

## Interacting with Charts

### Hover Effects

- Hover over data points to see detailed information
- Tooltips display exact values and percentages
- Additional context may be shown

### Click Actions

- Click on chart elements to filter data
- Drill down into specific time periods
- View detailed breakdowns

### Zoom and Pan

Some charts support:
- **Zoom**: Focus on specific time periods
- **Pan**: Navigate through time ranges
- **Reset**: Return to default view

## Customizing Charts

### Time Periods

Select different time periods to view:
- **Today**: Current day's data
- **Last 7 Days**: Past week
- **Last 30 Days**: Past month
- **Custom Range**: Select specific dates

### Filters

Apply filters to charts:
- **Campaign Filter**: View specific campaigns
- **Source Filter**: Filter by lead source
- **Status Filter**: Filter by lead or deal status

### Export Options

Some charts may support:
- **Export as Image**: Save chart as PNG or JPEG
- **Export Data**: Download underlying data as CSV
- **Print**: Print chart for reports

## Best Practices

1. **Regular Review**: Check charts regularly to spot trends
2. **Compare Periods**: Compare current period with previous periods
3. **Focus on Trends**: Look for patterns rather than single data points
4. **Take Action**: Use insights to optimize your outreach
5. **Document Insights**: Note important findings for future reference

## Troubleshooting

### Charts Not Loading

If charts don't load:
- Check your internet connection
- Refresh the page
- Clear browser cache
- Contact support if issues persist

### Missing Data

If charts show no data:
- Verify you have data for the selected time period
- Check filters aren't excluding all data
- Ensure campaigns have been created and executed
- Verify leads and deals exist in your account

### Incorrect Data

If charts show incorrect data:
- Refresh the dashboard
- Check data filters
- Verify time period selection
- Contact support if discrepancies persist
