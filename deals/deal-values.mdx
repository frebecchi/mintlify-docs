## Deal Values Overview

Deal values represent the monetary worth of sales opportunities. Deals can have single or multiple values with different currencies and revenue types.

## Single Value Deals

### Simple Deals

Most deals have a single value:
- One amount
- One currency
- One revenue type (one-time, monthly, etc.)

**Example:**
- Value: $5,000
- Currency: USD
- Revenue Type: One-time

## Multiple Value Deals

### Complex Deals

Deals can have multiple values:
- Multiple revenue streams
- Different currencies
- Mix of one-time and recurring
- Total calculated automatically

**Example:**
- Value 1: $1,000 one-time (USD)
- Value 2: $100 monthly (USD)

- Total: Sum of all values

## Revenue Types

### One-Time

Single payment:
- Paid once
- No recurring
- Common for services
- Project-based

### Monthly (MRR)

Monthly recurring revenue:
- Paid monthly
- Subscription model
- Recurring payments
- Tracked as MRR

### Quarterly

Quarterly recurring:
- Paid every 3 months
- Quarterly subscriptions
- Recurring revenue

### Annually

Annual recurring:
- Paid yearly
- Annual subscriptions
- Large upfront payments

## Managing Values

### Adding Values

In edit mode:
1. Click **"Add Value"** button
2. Enter amount
3. Select currency
4. Choose revenue type
5. Value added to deal

### Editing Values

Modify existing values:
- Change amount
- Update currency
- Modify revenue type
- Remove values

### Value Display

Values are displayed:
- In deal cards (total)
- In deal details (all values)
- In pipeline statistics
- In deal reports

## Currency Support

### Supported Currencies

- **USD**: US Dollar
- **EUR**: Euro
- **BRL**: Brazilian Real

### Currency Display

- Each value can have different currency
- Totals calculated per currency
- Displayed with currency symbol
- Can mix currencies in one deal

## Best Practices

1. **Accurate Values**: Use realistic deal values
2. **Appropriate Type**: Choose correct revenue type
3. **Track MRR**: Use monthly type for subscriptions
4. **Multiple Streams**: Use multiple values for complex deals
5. **Update Regularly**: Keep values current as deals progress
