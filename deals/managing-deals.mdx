## Creating Deals

### Manual Creation

Create a deal manually:
1. Click **"New Deal"** button
2. Enter deal title (required)
3. Set deal value (optional)
4. Select currency
5. Deal created in default stage

### From Campaigns

Deals can be created automatically:
- When leads respond to campaigns
- Based on campaign rules
- Manual creation from lead details
- Linked to source campaign

## Editing Deals

### Deal Details Sidebar

Click on a deal card to open details sidebar showing:
- Deal title
- Current stage
- Deal values
- Custom fields
- Source information

### Edit Mode

Enter edit mode:
1. Click **"Edit"** button
2. Modify deal information
3. Update custom fields
4. Change deal values
5. Save changes

### Deal Values

Edit deal values:
- Add multiple values
- Set currency per value
- Choose revenue type
- Calculate totals automatically

## Moving Deals

### Drag and Drop

Move deals on pipeline board:
1. Click and hold deal card
2. Drag to target stage
3. Drop in new column
4. Deal stage updates

### Stage Dropdown

Change stage from deal details:
1. Open deal details
2. Select new stage from dropdown
3. Deal moves to new stage
4. Updates immediately

## Deleting Deals

### Single Deal

Delete a deal:
1. Open deal details
2. Click **"Delete Deal"** button
3. Confirm deletion
4. Deal is permanently removed

<Warning>
  Deleting a deal cannot be undone. Make sure you want to delete before confirming.
</Warning>

## Deal Values

### Single Value

Simple deals with one value:
- One amount
- One currency
- One revenue type

### Multiple Values

Complex deals with multiple values:
- Multiple revenue streams
- Different currencies
- Mix of types
- Total calculated automatically

### Editing Values

In edit mode:
- Add new values
- Remove values
- Modify amounts
- Change currencies
- Update revenue types

## Custom Fields

### Viewing Fields

Custom fields appear in deal details:
- Field labels and values
- Read-only or editable
- Based on field configuration

### Editing Fields

In edit mode:
- Update field values
- Fill empty fields
- Change existing values
- Save changes

## Organization View

### My Deals

View only deals you created. Default view.

### All Organization Deals

View deals from all organization members:
- Available to admins and owners
- See all organization deals
- Filter by creator
- Note: Editing depends on ownership

## Best Practices

1. **Keep Updated**: Regularly update deal stages and values
2. **Use Custom Fields**: Track important deal information
3. **Set Realistic Values**: Use accurate deal values
4. **Move Regularly**: Update stages as deals progress
5. **Document**: Add notes via custom fields
