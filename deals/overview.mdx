## What are Deals?

Deals represent sales opportunities in your pipeline. Track deals through customizable stages from initial contact to closed won or lost.

## Deal Components

### Basic Information

**Deal Title:**
- Name identifying the deal
- Usually company or opportunity name
- Shown in pipeline cards

**Deal Value:**
- Monetary value of the deal
- Can include multiple values
- Supports different currencies
- Tracks one-time and recurring revenue

**Currency:**
- Currency for deal value
- USD, EUR, BRL supported
- Can be changed per deal

**Revenue Type:**
- One-time payment
- Monthly recurring (MRR)
- Quarterly recurring
- Annually recurring

### Pipeline Stage

**Current Stage:**
- Stage the deal is currently in
- Moves through pipeline stages
- Customizable stages
- Visual pipeline board

**Stage Information:**
- Stage name
- Stage color
- Stage order
- Default stage

### Custom Fields

Deals support custom fields:
- Track additional information
- Stage-specific fields
- Deal-specific data
- Custom field types

### Source Information

**Source Campaign:**
- Campaign that generated the deal
- Links deal to campaign
- Tracks lead source
- Campaign performance tracking

## Pipeline View

### Kanban Board

Deals are displayed in a Kanban-style board:
- Columns represent pipeline stages
- Cards represent individual deals
- Drag and drop to move deals
- Visual progress tracking

  Screenshot placeholder: Add a screenshot showing the Kanban pipeline board

### Pipeline Stages

**Default Stages:**
- New/Qualification
- Proposal
- Negotiation
- Closed Won
- Closed Lost

**Customizable:**
- Add custom stages
- Reorder stages
- Set stage colors
- Define default stage

### Deal Cards

Each deal card shows:
- Deal title
- Deal value
- Key custom fields
- Current stage
- Creator information (if organization)

## Deal Statistics

### Overview Cards

**Total Deals:**
- All deals in pipeline
- Includes all stages
- Shows total count

**Active Deals:**
- Deals in progress
- Excludes won/lost
- Currently active

**Won Deals:**
- Successfully closed deals
- Total count
- Total value (MRR and one-time)

**Lost Deals:**
- Deals closed without success
- Total count
- Total value lost

**Win Rate:**
- Percentage of won vs total closed
- Conversion rate
- Performance metric

**Total MRR:**
- Monthly recurring revenue
- From all active deals
- Pipeline value

## Deal Management

### Creating Deals

Create deals:
- Manually from deals page
- Automatically from campaigns
- Via API integration
- Import from external sources

### Moving Deals

Move deals through pipeline:
- Drag and drop on board
- Select stage from dropdown
- Update via deal details
- Bulk move operations

### Editing Deals

Edit deal information:
- Update title and value
- Change custom fields
- Modify stage
- Update source information

### Deleting Deals

Remove deals:
- Delete individual deals
- Bulk delete operations
- Permanent removal
- Cannot be undone

## Deal Values

### Single Value

Simple deals with one value:
- One-time payment
- Single amount
- One currency

### Multiple Values

Complex deals with multiple values:
- Multiple revenue streams
- Different currencies
- Mix of one-time and recurring
- Total calculated automatically

## Custom Fields

### Deal-Specific Fields

Track additional information:
- Company information
- Decision makers
- Timeline
- Budget
- Competitors
- Notes

### Field Types

- Text
- Number
- Date
- Dropdown
- Multiselect
- Boolean
- Textarea

### Stage-Specific Fields

Fields can be:
- Global (all stages)
- Stage-specific
- Conditional display

## Organization View

### My Deals

View only deals you created. This is the default view.

### All Organization Deals

View deals from all organization members. Available to:
- Organization owners
- Organization admins

**Features:**
- See all organization deals
- Filter by creator
- View organization-wide statistics
- Note: Editing depends on ownership

## Best Practices

1. **Keep Updated**: Regularly update deal stages and values
2. **Use Custom Fields**: Track important deal information
3. **Set Realistic Values**: Use accurate deal values
4. **Move Regularly**: Update stages as deals progress
5. **Document**: Add notes and custom fields
6. **Track Sources**: Link deals to campaigns
