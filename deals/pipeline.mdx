## Pipeline Overview

The deal pipeline is displayed as a Kanban-style board where each column represents a pipeline stage and each card represents a deal.

## Pipeline Stages

### Default Stages

**New/Qualification:**
- Initial stage for new deals
- Qualification and discovery
- First contact stage

**Proposal:**
- Proposal sent stage
- Negotiation begins
- Active discussions

**Negotiation:**
- Finalizing terms
- Contract discussions
- Closing process

**Closed Won:**
- Successfully closed deals
- Won opportunities
- Final stage for success

**Closed Lost:**
- Deals closed without success
- Lost opportunities
- Archived deals

### Customizing Stages

You can customize pipeline stages:
- Add custom stages
- Remove stages (except required)
- Reorder stages
- Set stage colors
- Define default stage

## Pipeline Board

### Column View

Each stage appears as a column:
- Stage name at top
- Deal count in column
- Total value in column
- Deal cards below

### Deal Cards

Each deal appears as a card showing:
- Deal title
- Deal value
- Key custom fields (up to 2)
- Current stage indicator
- Creator information (if organization)

### Drag and Drop

Move deals between stages:
1. Click and hold deal card
2. Drag to target stage column
3. Drop in new stage
4. Deal moves automatically

## Pipeline Actions

### Move Deal

Change deal stage:
- Drag and drop on board
- Select stage from dropdown (in deal details)
- Update via deal edit
- Bulk move operations

### View Deal Details

Click on deal card to:
- See full deal information
- Edit deal details
- Update custom fields
- Change deal value
- View deal history

### Create Deal

Add new deal:
1. Click **"New Deal"** button
2. Enter deal title
3. Set initial value (optional)
4. Deal created in default stage
5. Can move to appropriate stage

## Pipeline Statistics

### Stage Statistics

Each column shows:
- **Deal Count**: Number of deals in stage
- **Total Value**: Sum of all deal values in stage
- **Percentage**: Percentage of total pipeline

### Overall Statistics

Pipeline overview shows:
- Total deals across all stages
- Total pipeline value
- Average deal value
- Stage distribution

## Filtering Pipeline

### Active Pipeline

View active deals:
- Excludes Closed Won
- Excludes Closed Lost
- Shows deals in progress
- Default view

### Archived

View archived deals:
- Shows Closed Lost deals
- Archived opportunities
- Historical deals
- Separate tab/view

### Filters

Filter deals by:
- **Status**: Active, Won, Lost
- **Stage**: Specific pipeline stage
- **Date Range**: Created date filter
- **Search**: Deal title or source

## Best Practices

1. **Keep Updated**: Regularly move deals through stages
2. **Use Stages Appropriately**: Follow your sales process
3. **Track Progress**: Monitor stage progression
4. **Set Realistic Values**: Use accurate deal values
5. **Review Regularly**: Check pipeline health
