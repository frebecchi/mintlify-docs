## Creating a Sequence

To create a sequence, click **"Create Sequence"** from the sequences page.

## Step-by-Step Guide

### Step 1: Basic Information

#### Sequence Name

Enter a descriptive name:
- Identify sequence purpose
- Easy to remember
- Helpful for team members

**Examples:**
- "Welcome Sequence"
- "Follow-up Sequence"
- "Product Launch Sequence"

### Step 2: Add Steps

#### Message Steps

Add messages to your sequence:

1. Click **"Add Message"** or **"Add Step"**
2. Select "Message" step type
3. Enter message content
4. Use variables for personalization
5. Add formatting as needed

**Message Content:**
- Plain text
- Variable placeholders
- Emojis and special characters
- Line breaks

**Variables Available:**
- `{first_name}`: Lead's first name
- `{last_name}`: Lead's last name
- `{username}`: Telegram username
- `{phone_number}`: Phone number
- Custom field variables

#### Delay Steps

Add delays between messages:

1. Click **"Add Delay"** or **"Add Step"**
2. Select "Delay" step type
3. Set duration:
   - Minutes
   - Hours
   - Days
4. Configure delay amount

**Common Delays:**
- 24 hours: Daily follow-up
- 3 days: Weekly check-in
- 1 week: Monthly touchpoint

### Step 3: Configure Steps

#### Message Configuration

For each message step:
- Edit message text
- Add variables
- Format content
- Preview message

#### Delay Configuration

For each delay step:
- Set duration
- Choose unit (minutes/hours/days)
- Verify timing

### Step 4: Order Steps

Arrange steps in sequence:
- Drag to reorder
- Sequential execution
- Message → Delay → Message pattern
- Logical flow

### Step 5: Save Sequence

Review your sequence:
- Check all steps
- Verify delays
- Test variables
- Save sequence

## Sequence Examples

### Simple Welcome Sequence

**Step 1: Message**
```
Hi {first_name}! Welcome to our service. We're excited to have you.
```

**Step 2: Delay**
- 24 hours

**Step 3: Message**
```
Hey {first_name}, how are you finding our service so far? Let us know if you have any questions!
```

**Step 4: Delay**
- 3 days

**Step 5: Message**
```
Hi {first_name}, we'd love to hear your feedback. Your opinion matters to us!
```

### Follow-up Sequence

**Step 1: Message**
```
Hi {first_name}, I wanted to follow up on our previous conversation.
```

**Step 2: Delay**
- 2 days

**Step 3: Message**
```
Hey {first_name}, are you still interested? I'm here to help answer any questions.
```

**Step 4: Delay**
- 5 days

**Step 5: Message**
```
Hi {first_name}, this will be my last message. Feel free to reach out if you change your mind!
```

## Tips for Creating Sequences

1. **Start Simple**: Begin with 2-3 messages
2. **Space Appropriately**: Don't send messages too close together
3. **Personalize**: Use variables to personalize messages
4. **Test Content**: Review message content before saving
5. **Consider Timing**: Think about when leads will receive messages

## Best Practices

1. **Clear Purpose**: Each sequence should have a clear purpose
2. **Appropriate Length**: Don't make sequences too long
3. **Good Timing**: Space messages appropriately
4. **Personalization**: Use variables effectively
5. **Test First**: Test sequences before using in campaigns
