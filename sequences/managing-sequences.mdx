## Managing Sequences

Once sequences are created, you can manage them through various actions.

## Viewing Sequences

### Sequence List

The sequences page shows:
- Sequence name
- Number of steps
- Created date
- Updated date
- Actions menu

### Sequence Details

Click on a sequence to view:
- All steps in order
- Message content
- Delay durations
- Step configuration

## Editing Sequences

### Modify Sequence

Edit sequence content:
1. Click on sequence name
2. Click **"Edit"** button
3. Modify steps:
   - Update message text
   - Adjust delays
   - Add new steps
   - Remove steps
   - Reorder steps
4. Save changes

### Add Steps

Add new steps to sequence:
1. Open sequence for editing
2. Click **"Add Step"**
3. Select step type (Message or Delay)
4. Configure step
5. Position in sequence
6. Save

### Remove Steps

Delete steps from sequence:
1. Open sequence for editing
2. Click remove/delete on step
3. Confirm removal
4. Step is removed
5. Save changes

### Reorder Steps

Change step order:
1. Open sequence for editing
2. Drag steps to reorder
3. Or use up/down arrows
4. Save new order

## Duplicating Sequences

### Create Copy

Duplicate a sequence:
1. Select sequence
2. Click **"Duplicate"**
3. New sequence created with:
   - Same name + "(Copy)"
   - Same steps
   - Status: Draft
4. Edit copy as needed

<Info>
  Duplicating is useful for creating variations of successful sequences.
</Info>

## Deleting Sequences

### Remove Sequence

Delete a sequence:
1. Select sequence
2. Click **"Delete"**
3. Confirm deletion
4. Sequence is permanently removed

<Warning>
  Deleting a sequence doesn't affect campaigns already using it, but you won't be able to use it in new campaigns.
</Warning>

## Using Sequences in Campaigns

### Select Sequence

When creating campaigns:
1. Choose sequence option
2. Select sequence from dropdown
3. Sequence steps become campaign flow
4. Can customize per campaign

### Sequence Benefits

- Quick campaign setup
- Consistent messaging
- Reusable templates
- Easy updates

## Best Practices

1. **Regular Updates**: Keep sequences current
2. **Test Changes**: Test sequence modifications
3. **Document Purpose**: Add descriptions explaining sequence use
4. **Version Control**: Use naming to track versions
5. **Review Performance**: Monitor sequence effectiveness

## Troubleshooting

### Cannot Edit Sequence

If you cannot edit:
1. Check permissions
2. Verify sequence isn't locked
3. Try refreshing page
4. Check for errors

### Sequence Not Appearing

If sequence doesn't appear:
1. Check filters
2. Verify sequence exists
3. Refresh page
4. Check permissions
