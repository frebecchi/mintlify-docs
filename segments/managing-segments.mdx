## Managing Segments

Once segments are created, you can manage them through various actions and settings.

## Viewing Segments

### Segment List

The segments page displays all segments showing:
- Segment name and description
- Member count
- Active/Inactive status
- Rules summary
- Color indicator
- Actions menu

### Segment Details

Expand a segment to view:
- Full rule list
- Member list
- Member details
- Segment statistics

## Editing Segments

### Modify Segment

Edit segment configuration:
1. Click **"Edit"** on a segment
2. Update name, description, or color
3. Add, remove, or modify rules
4. Save changes
5. Segment refreshes automatically

### Update Rules

Modify segment rules:
- Add new rules
- Remove existing rules
- Change rule values
- Update operators
- Reorder rules (if applicable)

### Change Status

Activate or deactivate segments:
- Toggle Active/Inactive status
- Inactive segments preserve members
- Reactivate to resume updates

## Refreshing Segments

### Manual Refresh

Refresh segment membership:
1. Click **"Refresh"** button
2. Segment recalculates membership
3. Updates member count
4. Applies latest rules

<Info>
  Refresh segments after making changes to lead data or custom fields to ensure accurate membership.
</Info>

### Automatic Refresh

Segments automatically refresh:
- When lead data changes
- Periodically in background
- After rule changes
- On demand

## Viewing Members

### Expand Segment

View segment members:
1. Click expand arrow on segment
2. Member list appears
3. Shows all leads in segment
4. Can search and filter members

### Member Information

For each member, see:
- Contact information
- Custom field values
- Why they match segment
- Campaign participation

## Deleting Segments

### Remove Segment

Delete a segment:
1. Click **"Delete"** on segment
2. Confirm deletion
3. Segment is permanently removed
4. Leads are not deleted

<Warning>
  Deleting a segment removes it from all campaigns using it. Make sure no active campaigns depend on the segment.
</Warning>

## Organization View

### My Segments

View only segments you created. Default view.

### All Organization Segments

View segments from all organization members:
- Available to admins and owners
- See all organization segments
- Filter by creator
- Note: Editing depends on ownership

## Best Practices

1. **Regular Refresh**: Refresh segments after data changes
2. **Monitor Size**: Watch segment size for performance
3. **Update Rules**: Keep rules current with needs
4. **Test Changes**: Verify rule changes work correctly
5. **Document Purpose**: Use descriptions to explain segments

## Troubleshooting

### Segment Not Updating

If segment doesn't update:
1. Check segment is active
2. Verify rules are correct
3. Manually refresh segment
4. Check lead data matches rules

### Wrong Members

If segment has wrong members:
1. Review rules carefully
2. Check rule values
3. Verify operators
4. Test with single rule
5. Refresh segment

### Performance Issues

If segment is slow:
1. Check segment size
2. Review rule complexity
3. Consider splitting into multiple segments
4. Optimize rules
5. Contact support if needed
