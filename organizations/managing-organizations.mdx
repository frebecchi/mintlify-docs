## Managing Members

### Inviting Members

Invite team members:
1. Go to organization settings
2. Click **"Invite Member"**
3. Enter email address
4. Select role (Admin or Member)
5. Send invitation

**Invitation Process:**
- Invitation email sent
- Member accepts invitation
- Member joins organization
- Can access organization data

### Managing Members

As owner or admin:
- **View Members**: See all organization members
- **Change Roles**: Update member roles
- **Remove Members**: Remove from organization
- **Resend Invitations**: Resend pending invitations

### Member Roles

**Change Role:**
1. Find member in list
2. Click role dropdown
3. Select new role
4. Role updated

**Remove Member:**
1. Find member in list
2. Click remove/delete
3. Confirm removal
4. Member removed from organization

## Switching Organizations

### Organization Switcher

Use switcher to:
- Switch between organizations
- Access personal workspace
- View current organization
- Quick organization access

### Personal Workspace

Your personal workspace:
- Always available
- Not part of organization
- Your own data only
- Independent

## Best Practices

1. **Clear Names**: Use descriptive organization names
2. **Appropriate Roles**: Assign roles based on needs
3. **Regular Review**: Review members and roles regularly
4. **Document Purpose**: Add descriptions explaining organization purpose
5. **Manage Access**: Control who has admin access

## Troubleshooting

### Cannot Create Organization

If you cannot create:
1. Check account permissions
2. Verify account status
3. Try refreshing page
4. Contact support if needed

### Invitation Not Received

If member doesn't receive invitation:
1. Check email address is correct
2. Check spam folder
3. Resend invitation
4. Verify email is valid

### Cannot Manage Members

If you cannot manage members:
1. Verify you're owner or admin
2. Check organization permissions
3. Try refreshing page
4. Contact support if needed
